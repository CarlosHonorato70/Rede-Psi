# Rede Psi

Mental Health Social Network - Uma rede social focada em saúde mental.

## Estrutura do Projeto

### Backend
- **Localização**: `backend/`
- **Tecnologias**: Node.js, Express, MongoDB
- **Porta**: 5000

### Frontend
- **Localização**: `frontend/`
- **Tecnologias**: React, React Router
- **Porta**: 3000 (desenvolvimento)

## Como Executar

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```
